¿Qué es todo esto?

¿Qué es el desarrollo web? Desarrollo web significa construir y mantener sitios web; es el trabajo que tiene lugar en un segundo plano y que permite que una web tenga una apariencia impecable, un funcionamiento rápido y un buen desempeño para permitir la mejor experiencia de usuario. Los desarrolladores web son como duendes con poderes: nunca los ves, pero son los que hacen que todo esté bien y funcione de manera rápida y eficiente.

PHP

Es un lenguaje enfocado en la creación de webs dinámicas. Sus scripts son interpretados por el servidor y genera código HTML. Requiere Apache o IIS con librerías de PHP. Hereda su sintaxis de C, Java y Perl.Como principales ventajas, hemos de decir que es un lenguaje fácil de aprender y muy rápido. Soporta la orientación a objetos y utiliza un lenguaje multiplataforma. Además, puede conectarse con una gran cantidad de base de datos: MySQL, PostgreSQL, Oracle, MS SQL Server [1]

Javascript

Se utiliza principalmente del lado del cliente aunque se puede utilizar del lado del servidor. Actualmente y gracias a tecnologías como AJAX es utilizado para enviar y recibir información del servidor. Como principales ventajas, tenemos que destacar que es un lenguaje de scripting seguro y fiable, cuyos scripts tienen capacidades limitadas, debido a la seguridad.[1]

Botón de adorno

La web es parte de internet, pero no todo internet es la web. En pocas palabras, internet es una red mundial descentralizada de computadoras y servidores [2]

GET

El método GET se emplea para leer una representación de un resource

POST

Aunque se puedan enviar datos a través del método GET, en muchos casos se utiliza POST por las limitaciones de GET

PUT

Utilizado normalmente para actualizar contenidos, pero también pueden crearlos. Tampoco muestra ninguna información en la URL

DELETE

Simplemente elimina un resource identificado en la URI.

HEAD

Es idéntido a GET, pero el servidor no devuelve el contenido en el HTTP response. [3]

Cuando se quieren mandar datos como contraseñas se usa el metodo post puesto que van de alguna forma encriptadas Si se quiere acceder a una pagina por medio del url se usa el metodo get ya que pide información pública al servidor

CODIGOS DE ESTADO HTTP

Si algun dia un servidor regresa el codigo 200, no hay problema pues es un estado de ok

A veces cuando un usuario intenta acceder a un recurso prohibido, se le manda un error de no encontrado y el desarrollador esta en todo su derecho para delimitar el acceso

Si un dia te encuentras con el codigo 500 es porque es un código comúnmente emitido por aplicaciones empotradas en servidores web, mismas que generan contenido dinámicamente, por ejemplo aplicaciones montadas en IIS o Tomcat, cuando se encuentran con situaciones de error ajenas a la naturaleza del servidor web. [4]

Cuando veas que un atributo de HTML5 está despreciado es porque con las actualizaciones de versión, en ciertos navegadores o versiones del mismo lenguaje, ese atributo se pasa a la cola de prioridad y se ignora si se es posible, dandole prioridad de visualización a otros atributos

DIFERENCIAS ENTRE HTML5 Y HTML4

Se definió una sintaxis nueva en la que se incluyen cosas como etiquetas ()

Se simplifia el DOCTYPE, además de que permite incluir elementos SVG y MathML. Se incluyen igualmente cambios de atributos tradicionales y la desaparición de otros. [4]

Tablas y Formas

Para construir una tabla, se debe de tener en cuenta que se conforma por filas y columnas agrupadas, hablando de su estructura, y que se le pueden aplicar márgenes internos o respecto a otras secciones para poder construirlos

Las formas son como formularios a llenar en html5 que se pueden comprobar de acuerdo a los datos que se les ingresan
Algunos de esos controles pueden ser numéricos, de fecha, semánticos o de correo electrónico
